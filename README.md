# Docker Monitoring Agent

Этот агент автоматизирует мониторинг и анализ состояния Docker-окружения. Он собирает метрики по контейнерам (CPU, память, сетевой и дисковый I/O) и информации о системном использовании дискового пространства, анализирует данные и предоставляет структурированный отчёт с выявленными проблемами и рекомендациями по оптимизации.

## Архитектура

Агент построен на основе конечного автомата (архитектура ReAct) с пятью основными узлами:

1. **Планировщик** — формирует задачи для сбора метрик на основе пользовательского запроса.  
2. **Анализатор контейнеров** — получает список контейнеров и собирает детальную статистику по каждому.  
3. **Анализатор системы** — запрашивает общую информацию об использовании дискового пространства (образы, контейнеры, тома, кэш сборки).  
4. **Синтезатор** — объединяет все данные в единый отчёт и решает нужно ли ещё что-то.  

Агент использует строго типизированные Pydantic-модели для представления данных и поддерживает повторные попытки выполнения операций при сбоях. Результат работы — структурированный отчёт, пригодный для автоматической обработки или ручного анализа администратором.
